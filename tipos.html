Tipos de Servidores Web


En esta investigación vi demasiados servidores WEB, entre ellos están los siguientes:
.

I. Servidor HTTP Apache


Este servidor es demasiado conocido para los que están metidos en esto de los servidores, sinceramente yo no lo conocía, pero este servidor funciona de manera gratuita y es multiplataforma ya que funciona tanto para Linux, IOS, Unix, Windows, entre otros.
.

II. Servidor Nginx


Pues es igual al Apache ya que es multiplataforma compatible con Window, IOS, Linux, etc. Creo la única diferencia al Apache sería que este no solo tiene la versión gratuita, sino también tiene su versión de pago. Este servidor es conocido por su buen desempeño en cuanto a la gestión de visitas de usuarios al mismo tiempo, haciendo que los usuarios no vean lentas las cargas de sus páginas, aunque estas estén saturadas de usuarios. Actualmente también es uno de los servidores mas usados en el mundo.




III. LiteSpeed


Este servidor fue creado para suplantar al servidor Apache, mejorando el rendimiento del servidor WEB hablando en el ámbito de lo gráfico. Al igual que Ningx tiene tanto versión gratuita como de Pago.
.

IV. Servidor Microsoft IIS


Como vemos en su nombre, este servidor es desarrollado por Microsoft. La ventaja de este servidor es que puede ser ejecutado desde Windows gracias a la tecnología IIS y es compatible con muchas páginas programadas como .NET.
