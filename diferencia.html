Diferencias entre HTTP y HTML


La primera diferencia que encontré al investigar es que HTTP es un Protocolo de Comunicación, mientras que HTML es un formato de archivos como por ejemplo una imagen (“.jpg”, “.png”, entre otros…) o un documento (“.doc”, “.pdf”, entre otros…). En pocas palabras el HTTP sirve para acceder a paginas web mientras que HTML es un lenguaje que se usa para codificar información. Entonces podremos decir que HTML nos serviría para creas un recurso en internet mientras que el HTTP se usa para acceder a dicho recurso. En resumen, un Servidor puede no solo ser una maquina gigante que almacena mucha información, sino que también puede ser el celular o la computadora que estamos usando actualmente para acceder a este documento o a la plataforma en la que publicamos nuestras tareas.
